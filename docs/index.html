🔑 Provider Setup

  • OpenRouter (recommended) — free tier available
    • Get a free API key at openrouter.ai/keys
    • Paste in Settings → OpenRouter → Save
    • Select a model — free ones have :free suffix
    • Check "Use OpenRouter as active provider"
  • Ollama Cloud — local + cloud models
    • Requires Ollama Cloud API key for ☁️ models
    • Local models work without a key
  • LM Studio — local OpenAI-compatible endpoint
  • Cloud Direct — Gemini, OpenAI, Claude, Azure (browser API calls)
